La herramienta Laravel cuenta con una serie de características con las que los desarrolladores puede construir sus aplicaciones web de una forma rápida y sencilla. En esta lista se muestran algunas de las más importantes.

Principales características

  • Sistema de enrutamiento, también RESTful
  • Soporte para el caché
  • Soporte para MVC10
    • Librerías y modularidad
    • Seguridad mediante librerías como BCrypt
  • Gestión de sesiones
  • Sistema de autenticación
  • Un sistema para creación de colas de trabajo
  • La posibilidad de acceder a datos en realtime
